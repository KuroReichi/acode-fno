# File Name Only

A simple Acode plugin that hides file extensions in the file explorer, displaying only the file name.

## Features

- Hides the last file extension.
- Supports filenames with multiple dots.
- Automatically updates when files are added, renamed, or refreshed.
- Lightweight with no configuration required.

## Examples

| Original | Displayed |
|----------|-----------|
| `Database.js` | `Database` |
| `Database.java` | `Database` |
| `Command.Node.js` | `Command.Node` |
| `archive.tar.gz` | `archive.tar` |

## Installation

1. Download the latest release.
2. Install it from the Acode Plugin Manager.
3. Restart Acode if necessary.
